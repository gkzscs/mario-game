# 像素冒险家 (Pixel Adventurer)

经典平台跳跃风格游戏 - 个人学习项目

## 项目简介

这是一个使用 HTML5 Canvas + JavaScript 开发的平台跳跃游戏，用于学习游戏开发技术。

## 功能特性

- 平台跳跃游戏机制
- 关卡设计系统
- 敌人AI与碰撞检测
- 道具收集系统
- 得分与生命系统

## 技术栈

- **前端**: HTML5 Canvas + JavaScript (ES6+)
- **构建工具**: Vite
- **测试框架**: Jest

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build

# 运行测试
npm test
```

## 项目结构

```
mario-game/
├── docs/              # 设计文档
├── src/
│   ├── assets/        # 游戏资源（图片、音频）
│   ├── js/            # 游戏逻辑
│   ├── css/           # 样式文件
│   └── index.html     # 入口文件
├── tests/             # 测试文件
└── package.json
```

## 操作说明

- **← → 或 A D**: 移动
- **↑ 或 W 或 空格**: 跳跃
- **目标**: 收集金币，踩踏敌人，获得高分！

## 游戏资源来源

本项目使用的游戏素材为程序绘制的几何图形。
