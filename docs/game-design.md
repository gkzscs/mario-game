# 游戏设计文档

## 游戏概述

**游戏名称**: 像素冒险家 (Pixel Adventurer)

**游戏类型**: 2D平台跳跃游戏

**目标平台**: Web浏览器 (HTML5 Canvas)

**目标玩家**: 喜欢经典平台跳跃游戏的玩家

## 核心玩法

### 基本操作
- **移动**: 左右方向键 / A、D键
- **跳跃**: 上方向键 / W键 / 空格键
- **目标**: 收集金币，避开或踩踏敌人，到达终点

### 游戏机制

1. **跳跃系统**
   - 玩家可以跳跃到平台上
   - 重力系统使玩家自然下落
   - 踩踏敌人可以弹跳

2. **敌人系统**
   - 敌人在固定范围内巡逻
   - 从上方踩踏可以消灭敌人并得分
   - 从侧面碰撞会失去生命

3. **收集系统**
   - 金币散布在关卡中
   - 收集金币增加分数
   - 收集一定数量金币可能获得额外生命

4. **生命系统**
   - 初始3条生命
   - 碰到敌人或掉落失去生命
   - 生命归零游戏结束

## 关卡设计

### 第一关: 草原
- 教学关卡
- 简单的平台布局
- 少量敌人

### 第二关: 森林
- 更多跳跃挑战
- 移动平台

### 第三关: 城堡
- 高难度挑战
- 更多敌人类型

## 角色设计

### 玩家角色
- 红色帽子
- 黄色皮肤
- 红色衣服

### 敌人类型
1. **巡逻怪**: 在固定范围来回移动
2. **跳跃怪**: 会跳跃的敌人
3. **飞行怪**: 空中巡逻的敌人
